# 住宅ローン vs 賃貸比較

住宅購入と賃貸の総コストを、同じ居住年数でシンプルに比較する単一ページのWebアプリです。  
HTML / CSS / JavaScript のみで構成しており、GitHub Pages でそのまま公開できます。

## ファイル構成

- `index.html` : 画面構造
- `style.css` : レイアウトとレスポンシブ対応
- `script.js` : 入力処理、計算、結果表示

## 使い方

1. `index.html` をブラウザで開きます。
2. 家賃、居住年数、購入条件などを入力します。
3. 入力内容に応じて、購入総コスト・賃貸総コスト・差額が自動で更新されます。
4. `サンプル値を入力` ボタンで動作確認できます。
5. `リセット` ボタンで入力値を初期状態に戻せます。

## 計算の考え方

- 住宅ローンは元利均等返済で月返済額を計算します。
- 購入総コストは、ローン返済額、固定資産税、管理費、修繕積立金、購入時諸費用を合計し、将来売却価格を差し引いて算出します。
- 賃貸総コストは、家賃総額と更新料総額の合計です。
- 家賃上昇率を指定した場合は、年ごとに家賃が上がる前提で集計します。

## 注意

このツールは参考シミュレーションです。税制、保険、金利変動、将来の修繕費増加、売却時の手数料、不動産相場の変動などは簡略化しています。

## GitHub Pages で公開する方法

1. このフォルダをGitHubリポジトリに配置します。
2. GitHubの `Settings` → `Pages` を開きます。
3. 公開元のブランチとフォルダを選択します。
4. 数分待つと公開URLが発行されます。

`tools/home-rent-vs-buy/` をそのまま公開対象にするか、必要に応じてリポジトリの公開ルートへ配置してください。
