# 簡易間取り図作成ウェブアプリ

不動産実務向けの、ブラウザで動くシンプルな間取り図エディタです。SVG ベースで部屋や設備パーツを配置し、JSON 保存・再読込・PNG 出力まで行えます。

## ファイル構成

- `index.html`
- `style.css`
- `app.js`
- `README.md`

## 使い方

1. `index.html` をブラウザで開きます。
2. 起動直後に 1K の簡易テンプレートが表示されます。
3. 左側の追加ボタンから部屋やパーツを追加します。
4. 要素をクリックすると選択され、ドラッグで移動できます。
5. 部屋は右下のハンドルをドラッグしてリサイズできます。
6. 左側の入力欄で名称や表示テキストを編集できます。
7. 不要な要素は「選択要素を削除」または Delete キーで削除できます。

## 保存・読込方法

- `JSON保存` を押すと、現在の間取り図を JSON ファイルとして保存します。
- `JSON読込` を押して保存済み JSON を選ぶと、内容を再編集できます。

## PNG出力方法

- `PNG出力` を押すと、現在の SVG キャンバスを PNG 画像として保存します。
- 出力時は白背景つきで画像化されるため、そのまま資料のたたき台として利用できます。
